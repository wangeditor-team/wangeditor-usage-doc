# 插入网络视频的回调事件

使用 `editor.config.onlineVideoCallback` 可以自定义检查插入网络视频后的回调。

```javaScript
const E = window.wangEditor
const editor = new E('#div1')

// 自定义检查插入视频的回调
editor.config.onlineVideoCallback = function (video) {
  // 自定义回调内容，内容成功插入后会执行该函数
  console.log('插入视频内容', video)
}

editor.create()
```
